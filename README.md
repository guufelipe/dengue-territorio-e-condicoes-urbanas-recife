# dengue-territorio-e-condicoes-urbanas-recife
Neste projeto tenho o bjetivo de transformar dados abertos de saúde pública do Recife, especialmente casos de Dengue, Zika e Chikungunya, em informação acessível, visual e útil. A partir da análise territorial e temporal dos dados, busco identificar padrões, desigualdades urbanas e áreas prioritárias que demandam maior atenção.
